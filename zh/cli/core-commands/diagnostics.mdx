title: 环境诊断

---

> 跑不起来或登录异常时，先用这些命令排查。

## `octopus doctor`（检查你的电脑环境是否正常）

**什么意思？** 检查 Node.js、内置引擎、Chrome 可用性及浏览器模式等，排查「跑不起来」类问题。

**基本用法：**

```bash
octopus doctor
```

成功时返回 `"ok": true`。若某项为 `ok: false`，按提示修复后再运行采集任务。

**浏览器专项诊断：**

```bash
octopus browser doctor
```

## `octopus auth status`（查看登录状态）

**什么意思？** 查看当前 CLI 是否已登录、API Key 是否有效。

**基本用法：**

```bash
octopus auth status
```

脚本中可配合 `--json` 输出结构化结果。若未登录，请先 [创建 API Key](/zh/mcp/quick-start/api-key) 并执行 `octopus auth login`。
