# 社交媒体数据挖掘

> 团队如何采集公开社交帖子、个人资料、互动和趋势信号，同时遵守平台限制、隐私考虑和 API 替代方案。

## 采集内容

| 平台 | 数据类型 | 应用场景 |
|---|---|---|
| Twitter/X | 帖子、个人资料、话题标签、趋势 | 品牌监控、情感分析 |
| LinkedIn | 个人资料、公司页面、招聘信息 | B2B 线索生成 |
| Instagram | 帖子、个人资料、话题标签 | 网红研究 |
| Reddit | 帖子、评论、版块 | 趋势分析、情感分析 |

## 法律与伦理考量

<Warning>
  采集社交媒体数据存在重大的法律和伦理问题。许多平台在其服务条款中禁止数据采集。公开数据不等于可自由使用。GDPR 和 CCPA 适用于个人数据。在大规模社交媒体采集前请咨询法律顾问。
</Warning>

## 实用方法

1. **优先使用官方 API** — 大多数平台提供官方 API
2. **采集作为后备方案** — 仅在 API 不足时使用采集
3. **聚焦公开数据** — 个人资料名称、帖子、互动指标
4. **遵守速率限制** — 缓慢、尊重地采集可降低风险
5. **不得重新发布** — 将采集数据用于分析而非再分发
