# 如何采集 YouTube 数据

> 如何从 YouTube 提取视频元数据、频道数据、评论和搜索结果，用于竞品分析、内容研究和市场情报。

## 采集内容

| 数据类型 | 字段 |
|---|---|
| 视频元数据 | 标题、描述、观看数、点赞数、发布日期、时长 |
| 频道数据 | 频道名称、订阅数、总观看数、视频数 |
| 搜索结果 | 视频标题、频道、观看数、发布日期 |
| 评论 | 评论文本、作者、日期、点赞数、回复数 |

## 工作流

<Steps>
  <Step title="从搜索或频道页面开始">
    从搜索结果或指定频道采集视频列表。
  </Step>
  <Step title="提取元数据">
    收集标题、观看数、点赞数、描述和发布日期。
  </Step>
  <Step title="访问视频页面获取评论">
    从视频详情页提取评论和更多元数据。
  </Step>
  <Step title="处理翻页">
    采集搜索或频道页面中所有相关结果。
  </Step>
</Steps>

## 技术注意事项

- YouTube 使用重度 JavaScript 渲染；真实浏览器可提高可靠性。
- YouTube Data API v3 是访问 YouTube 数据的官方合规方式。
- API 有配额限制；相应规划数据采集。
- 通过 API 采集评论需要特殊权限。

## 常见应用场景

- **内容研究** — 了解某个品类中哪些视频表现良好
- **竞品监控** — 追踪竞品频道表现
- **情感分析** — 分析评论区中的品牌印象
- **SEO 研究** — 识别热门话题和搜索模式
