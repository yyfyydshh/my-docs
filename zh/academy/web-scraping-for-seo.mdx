# 利用网页采集进行 SEO

> 如何使用网页采集进行 SEO 研究、竞品分析、搜索优化和搜索排名追踪。

## 采集内容

| 数据类型 | 应用场景 |
|---|---|
| 竞品内容 | 分析竞品在哪些话题和格式上获得排名 |
| 关键词排名 | 跨关键词和地区追踪搜索排名 |
| 外链概况 | 监控新增和丢失的外链 |
| 网站结构 | 映射网站架构并识别可爬行性问题 |
| 页面元素 | 收集标题标签、H1 标签、Meta 描述和内容长度 |

## 常见工作流

### 竞品内容分析

<Steps>
  <Step title="识别竞品">
    找到在目标关键词上排名的网站。
  </Step>
  <Step title="收集内容">
    从竞品排名靠前的页面采集标题、描述、标题标签和内容。
  </Step>
  <Step title="分析模式">
    识别内容长度、关键词用法、格式和结构模式。
  </Step>
  <Step title="应用洞察">
    将发现用于指导你自己的内容策略。
  </Step>
</Steps>

### 排名追踪

追踪你的页面和竞品页面在目标关键词上跨搜索引擎和地区的排名。

### 技术 SEO 审计

- 查找死链和 404 页面
- 识别重复或薄内容
- 映射网站结构和内部链接
- 收集标题标签、Meta 描述和标题结构

## 工具与方法

- **排名追踪器** — SEMrush、Ahrefs、Moz 用于追踪排名（非采集方式）
- **网站爬虫** — Screaming Frog、Sitebulb 用于技术 SEO 分析
- **定制爬虫** — 用于超越标准工具范围的定制化竞品研究

## 重要说明

- 遵守 SEO 爬虫的 robots.txt 指令
- 不要直接采集 Google — 使用官方搜索 API 获取排名数据
- 专注于用分析创造价值，而非简单复制竞品内容
