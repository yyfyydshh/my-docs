# 如何采集 Facebook Marketplace 数据

> 如何从 Facebook Marketplace 提取商品列表、价格、卖家数据和位置信息，用于本地市场调研、线索生成和竞品分析。

## 采集内容

| 数据类型 | 字段 |
|---|---|
| 商品列表 | 标题、价格、类别、成色、描述 |
| 位置 | 城市、社区、距搜索中心的距离 |
| 卖家数据 | 卖家名称、主页链接、商品数量、回复率 |
| 图片 | 商品图片（用于产品识别） |

## 工作流

<Steps>
  <Step title="设置位置参数">
    使用城市、邮政编码或半径定义搜索区域。
  </Step>
  <Step title="采集列表数据">
    从搜索结果中提取标题、价格、类别、成色和位置。
  </Step>
  <Step title="访问列表详情页">
    提取完整描述、卖家信息和更多图片。
  </Step>
  <Step title="处理分类和搜索翻页">
    跨多个分类或搜索查询采集商品。
  </Step>
</Steps>

## 技术注意事项

- Facebook Marketplace 使用重度 JavaScript 渲染；需要真实浏览器。
- 部分内容或完整卖家主页可能需要登录。
- Facebook 有主动的反爬措施；使用适当的速率限制和会话管理。
- 位置数据是近似的（城市/社区级别，而非精确地址）。

## 常见应用场景

- **房地产市场调研** — 监控出租和出售房源
- **车辆市场分析** — 按品牌/型号/年份追踪车辆价格
- **本地产品研究** — 了解特定品类按地理位置的价格
