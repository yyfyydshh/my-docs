# 如何采集 Instagram 数据

> 如何在遵守平台政策和速率限制的前提下，从 Instagram 提取公开帖子、个人资料、互动指标和话题标签数据。

## 采集内容

| 数据类型 | 字段 |
|---|---|
| 公开帖子 | 图片/视频 URL、标题、话题标签、发布日期、点赞数、评论数 |
| 个人资料 | 用户名、简介、粉丝数、关注数、帖子数 |
| 话题标签数据 | 标签名称、帖子数、热门帖子 |
| 评论 | 评论文本、用户名、日期 |

## 法律与伦理考量

<Warning>
  Instagram 在其服务条款中禁止数据采集。使用官方 Instagram Graph API 是合规方式。采集可能导致账户被封禁或法律诉讼。在进行之前请咨询法律顾问。
</Warning>

## 技术注意事项

- Instagram 访问大部分数据需要真实浏览器和会话 Cookie。
- 平台会主动检测并阻止自动化访问。
- 考虑使用 Instagram Graph API 以适当权限合规地访问数据。
- 数据采集应仅聚焦公开信息。

## 实用方法

1. **使用 Instagram Graph API** — 以适当权限合规访问公开数据的官方方式
2. **非官方采集** — 法律和技术风险较高，可能违反服务条款
3. **第三方数据提供商** — 持有 Instagram 数据授权协议的公司
