# 什么是零代码网页采集？

> 零代码网页采集让非开发者无需编写 Python、XPath 或 HTTP 代码即可可视化地从网站提取数据。

## 可视化采集的工作原理

<Steps>
  <Step title="输入 URL">
    在工具内置浏览器中打开目标网站。
  </Step>
  <Step title="选取数据">
    点击元素将其选为字段。工具自动生成底层逻辑。
  </Step>
  <Step title="定义工作流">
    可视化地添加翻页、点击、滚动等操作。
  </Step>
  <Step title="运行与导出">
    在本地或云端执行，然后导出至你偏好的格式。
  </Step>
</Steps>

## 能力范围

- 从任何公开网站提取结构化数据
- 处理翻页、无限滚动和多页面工作流
- 按计划自动执行周期性数据采集
- 带代理轮换的云端提取
- 导出为 CSV、Excel、JSON、数据库和 API
- Amazon、LinkedIn、Google Maps 等预置模板

## 适用人群

- **分析师** — 需要数据但不想写代码
- **市场营销人员** — 监控竞品和市场趋势
- **研究人员** — 采集结构化数据集
- **小团队** — 没有专门工程资源
