# 网页采集术语表

> 网页采集行业常用术语和概念的参考指南。

## A

**API（应用程序编程接口）**
一组协议和工具，允许不同软件应用程序之间相互通信。

**ASP（应用服务提供商）**
通过互联网托管应用程序的第三方提供商。

## C

**爬取（Crawl）**
系统性地浏览和索引网页，通常用于构建搜索索引或发现内容。

**CSS 选择器**
用于选择和定位页面上特定 HTML 元素以进行数据提取的模式。

## H

**无头浏览器（Headless Browser）**
没有图形用户界面的浏览器，能够以编程方式渲染和交互网页。

## P

**翻页（Pagination）**
将内容分散到多个页面的做法，要求爬虫通过多个 URL 或交互进行导航。

**代理（Proxy）**
路由你请求的中间服务器，隐藏你的原始 IP 地址以避免被检测。

## R

**渲染（Render）**
执行 JavaScript 并生成真实浏览器将显示的最终 HTML 输出。

## S

**采集（Scraping）**
使用软件工具自动从网站收集数据的过程。

**选择器（Selectors）**
用于定位页面上特定元素以进行提取的模式（XPath、CSS）。

## X

**XPath**
一种从 XML 文档中选择节点的语言，常用于网页采集中以定位 HTML 元素。
