# 房地产数据采集

> 房地产团队如何从公开门户采集房源、租赁、成交、经纪人及社区数据，同时处理数据时效性、去重和合规问题。

## 房地产团队采集什么

| 数据类型 | 应用场景 | 典型字段 |
|---|---|---|
| 房源列表 | 库存监控 | 价格、地址、卧室数、浴室数、面积、图片、挂牌日期 |
| 租赁列表 | 租赁市场分析 | 租金、卧室数、浴室数、位置、配套设施、可租日期 |
| 成交数据 | 市场趋势分析 | 成交价、成交日期、在售天数、价格历史 |
| 经纪人数据 | 线索生成 | 经纪人姓名、经纪公司、电话、房源列表、评价 |

## 常见来源

- Zillow、Redfin、Realtor.com（美国）
- Rightmove、Zoopla（英国）
- 贝壳、安居客（中国）

## 追踪的关键指标

- **在售天数** — 需求强度指标
- **价格变动** — 议价信号与市场动态
- **每平方英尺价格** — 标准化对比指标
- **新增房源** — 市场活跃度
