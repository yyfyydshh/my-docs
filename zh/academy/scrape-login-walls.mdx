# 采集登录墙后的内容

> 爬虫如何处理带身份验证的页面：Cookie、会话复用、浏览器登录、MFA 限制及账户安全的操作实践。

## 方法

### 1. Cookie 与会话复用
登录一次，导出 Cookie，在爬虫中复用。

### 2. 浏览器自动化登录
使用 Playwright 或 Puppeteer 自动化登录流程、处理 MFA，然后提取会话 Cookie。

### 3. API 认证
OAuth 2.0 流程、基于 API Key 或 Token 的认证。最可靠且受到平台认可。

## 最佳实践

- **使用专用账户** — 不要使用个人账户进行采集
- **轮换会话** — 不要无限期地重复使用同一会话
- **谨慎处理 MFA** — 自动化 MFA 不可靠
- **监控账户问题** — 留意验证码或登录告警
- **遵守服务条款** — 违反服务条款登录可能导致账户被注销
