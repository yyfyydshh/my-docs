# 如何采集 LinkedIn 数据

> 采集 LinkedIn 公开个人资料、公司页面、招聘信息和帖子在技术和操作层面涉及的内容，以及法律和伦理保障。

## 可采集内容

| 数据类型 | 字段 | 复杂度 |
|---|---|---|
| 公司页面 | 名称、行业、规模、地点 | 中等 |
| 招聘信息 | 职位、公司、地点、薪资 | 中等 |
| 公开个人资料 | 姓名、简介、地点、当前职位 | 高 |

## 法律考量

<Warning>
  LinkedIn 对数据采集有激进的执法力度，并曾对采集者提起法律诉讼。商业用途请优先考虑官方 API 并咨询法律顾问。
</Warning>

## 采集的替代方案

1. **LinkedIn Sales Navigator** — 官方付费工具
2. **LinkedIn API** — 授权访问的官方 API
3. **第三方数据提供商** — Apollo、ZoomInfo 等持有授权协议
