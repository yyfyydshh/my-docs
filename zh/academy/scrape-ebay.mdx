# 如何采集 eBay 数据

> 如何从 eBay 提取产品列表、卖家数据、价格、评论及拍卖数据，用于竞品情报、价格监控和市场调研。

## 采集内容

| 数据类型 | 字段 |
|---|---|
| 产品列表 | 标题、价格、成色、形式（拍卖/一口价）、所在地 |
| 卖家数据 | 用户名、信用评分、所在地、注册时间 |
| 拍卖数据 | 当前出价、出价次数、剩余时间、一口价 |
| 搜索结果 | 所有匹配商品及其价格和卖家信息 |

## 工作流

<Steps>
  <Step title="从搜索结果开始">
    从分类或关键词搜索 URL 开始，采集广泛的商品列表。
  </Step>
  <Step title="提取列表数据">
    采集标题、当前价格、成色、形式和卖家信息。
  </Step>
  <Step title="访问详情页">
    提取完整描述、规格、配送选项和评论数据。
  </Step>
  <Step title="处理翻页">
    eBay 搜索结果有翻页；采集与你研究相关的所有商品。
  </Step>
</Steps>

## 技术注意事项

- eBay 使用 JavaScript 渲染；需要真实浏览器才能可靠提取。
- 翻页使用 `?_pgn=` 参数或跟随"下一页"链接。
- 卖家信用评分可从列表页或卖家主页提取。
- 大批量采集时注意速率限制。

## 常见应用场景

- **价格监控** — 追踪各品类的成交价和当前价
- **竞品研究** — 监控卖家提供什么商品及价格
- **拍卖情报** — 追踪出价模式和成交率
- **产品研究** — 在采购前验证产品需求
