本文档演示从自然语言到提取数据的完整 八爪鱼 MCP 工作流程。

## 第一步：发现模板

**请求：** *"帮我找一个采集 Amazon 商品列表的 八爪鱼 模板。"*

智能体调用 `search_templates` 来搜索相关的预置采集模板。

## 第二步：运行任务

**请求：** *"运行 Amazon 热销榜模板。"*

智能体调用 `execute_task` 来执行任务。MCP 服务端使用所选模板启动云端提取，完成后返回结果。

## 第三步：导出结果

**请求：** *"将结果导出为 CSV 格式。"*

智能体调用 `export_data`（指定 format=csv）返回完整数据集。

## 可用工具参考

| 工具 | 功能 |
|---|---|
| `search_templates` | 按关键词搜索预置采集模板 |
| `search_tasks` | 列出并搜索你已有的 八爪鱼 任务 |
| `start_or_stop_task` | 通过 ID 启动或停止云端任务 |
| `execute_task` | 运行采集模板并获取结果 |
| `export_data` | 将已完成任务的数据导出为 JSON 或 CSV |
| `redeem_coupon_code` | 兑换促销优惠码 |
