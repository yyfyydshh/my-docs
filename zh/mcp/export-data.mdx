---
title: export_data
description: 将已完成任务的数据导出为 JSON 或 CSV。
---

从已完成的 八爪鱼 任务运行中导出数据。

## 参数

| 参数 | 类型 | 说明 |
|---|---|---|
| `taskId` | string | 任务 ID |
| `runId` | string | 运行 ID（如任务有最近运行则可不填） |
| `format` | string | 导出格式：json 或 csv |
| `limit` | number | 最大返回记录数 |

## 响应

以请求的格式返回提取的数据。

另请参阅：[MCP 概览](/zh/mcp/index)、[MCP 工作流程](/zh/mcp/workflow)
