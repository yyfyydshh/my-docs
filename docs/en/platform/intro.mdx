# What is Octoparse?

> Octoparse is a no-code web data extraction platform built for analysts, growth teams, and operations people who need structured data from websites - without writing scrapers from scratch.

Octoparse helps you turn website content into structured data by building scraping workflows visually, running them locally or in the cloud, and exporting results to files, databases, cloud storage, or downstream systems.

## What Octoparse Does

<Card title="Build Scraping Workflows">
  Select data on a web page, define actions such as clicks and pagination, and turn the workflow into a reusable task.
</Card>

<Card title="Run Data Extraction">
  Execute tasks locally or in the cloud, depending on the website, task setup, and automation needs.
</Card>

<Card title="Export Structured Data">
  Send extracted results to CSV, Excel, JSON, Google Sheets, databases, or cloud storage.
</Card>

## How It Works

<Steps>
  <Step title="Open the Target Website">
    Start from a URL, template, or custom task. Octoparse loads the page in its built-in browser.
  </Step>
  <Step title="Define the Extraction Logic">
    Select data fields, then add actions like clicking links, handling pagination, scrolling, or logging in.
  </Step>
  <Step title="Run and Export the Task">
    Run locally or in the cloud, monitor progress, then export to your preferred format.
  </Step>
</Steps>

## Common Use Cases

- **Price Monitoring** - Track competitor prices in real time
- **Lead Generation** - Build targeted contact lists from public directories
- **Market Research** - Collect product catalogs, reviews, and ratings
- **E-commerce Data** - Extract listings, variants, seller data, and inventory
- **Real Estate** - Gather listing, rental, and agent data
- **Recurring Data Collection** - Automate ongoing extraction for reporting

<Note>
  The exact behavior of a task depends on the website structure, task settings, extraction mode, and whether the task runs locally or in the cloud.
</Note>
