# How to Scrape Instagram

> How to extract public Instagram posts, profiles, engagement metrics, and hashtag data while respecting platform policies and rate limits.

## What to Collect

| Data Type | Fields |
|---|---|
| Public posts | Image/video URL, caption, hashtags, post date, likes, comments |
| Profiles | Username, bio, follower count, following count, post count |
| Hashtag data | Hashtag name, post count, top posts |
| Comments | Comment text, username, date |

## Legal and Ethical Considerations

<Warning>
  Instagram prohibits scraping in its Terms of Service. Using the official Instagram Graph API is the compliant approach. Scraping may result in account bans or legal action. Consult legal counsel before proceeding.
</Warning>

## Technical Notes

- Instagram requires a real browser and session cookies for most data access.
- The platform actively detects and blocks automated access.
- Consider Instagram Graph API for compliant data access with appropriate permissions.
- Data collection should focus on public information only.

## Practical Approaches

1. **Use Instagram Graph API** - Official, compliant way to access public data with proper permissions
2. **Unofficial scraping** - Higher legal and technical risk, may violate ToS
3. **Third-party data providers** - Companies with licensing agreements for Instagram data
