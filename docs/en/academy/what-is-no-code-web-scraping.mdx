# What is No-Code Web Scraping?

> No-code web scraping lets non-developers extract data from websites visually - without writing Python, XPath, or HTTP code.

## How Visual Scraping Works

<Steps>
  <Step title="Enter a URL">
    Open the target website in the tool built-in browser.
  </Step>
  <Step title="Select data">
    Click on elements to select them as fields. The tool generates the underlying logic automatically.
  </Step>
  <Step title="Define workflow">
    Add pagination, clicks, scrolls, and other actions visually.
  </Step>
  <Step title="Run and export">
    Execute locally or in the cloud, then export to your preferred format.
  </Step>
</Steps>

## Capabilities

- Extract structured data from any public website
- Handle pagination, infinite scroll, and multi-page workflows
- Automate recurring data collection on a schedule
- Cloud extraction with proxy rotation
- Export to CSV, Excel, JSON, databases, and APIs
- Pre-built templates for Amazon, LinkedIn, Google Maps, and more

## Who It Is For

- **Analysts** who need data without writing code
- **Marketers** monitoring competitors and market trends
- **Researchers** collecting structured datasets
- **Small teams** without dedicated engineering resources
