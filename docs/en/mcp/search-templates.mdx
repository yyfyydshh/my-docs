# search_templates

Search for pre-built Octoparse scraping templates by keyword or category.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `query` | string | Search keyword or phrase |
| `limit` | number | Maximum number of results (default: 10) |

## Response

Returns a list of matching templates with name, description, target website, supported page types, and last updated date.
