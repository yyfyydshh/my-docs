# start_or_stop_task

Start or stop an Octoparse cloud extraction task by its ID.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `taskId` | string | The task ID to control |
| `action` | string | start or stop |

## Response

Returns task status and run ID (for start action).
