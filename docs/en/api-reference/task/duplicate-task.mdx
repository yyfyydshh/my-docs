# Duplicate Task

> **Required plan:** Standard, Professional, or Enterprise

## Endpoint

```
POST https://openapi.octoparse.com/task/duplicate
```

## Request Body

| Parameter | Type | Required | Description |
|---|---|---|---|
| `taskGroupId` | string | Yes | The task group ID to duplicate |
| `newName` | string | No | Name for the new task |

## Response

Returns the ID and name of the newly created task.
