# 根据指定位置获取任务批次数据

> **所需套餐：** 旗舰版、旗舰+、企业版、团队版

## 端点

```
https://openapi.bazhuayu.com/data/lotno/all
```


## 请求头

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `Authorization` | String | 是 | 身份令牌 |

## 响应

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `data` | Object | 是 | 响应数据 |
| `data` | Object | 是 | 任务数据 |
| `total` | Int | 是 | 数据总量 |
| `restTotal` | Int | 是 | 剩余数据 |
| `offset` | Int | 是 | 数据偏移量，当offset小等于0时，则从起始位置读取任务数据 |
| `requestId` | String | 是 | 请求Id |

## 错误响应

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `requestId` | String | 是 | 请求Id |
| `error` | Object | 是 | 错误信息，参考错误码 |
| `code` | String | 是 | 错误码 |
| `message` | String | 是 | 错误描述 |
