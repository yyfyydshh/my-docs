# 认证

> 所有 API 请求均需使用 Access Token 进行认证。

## Token 获取

调用 [获取新 Token](/en/api-reference/access-token/obtain-a-new-token) 接口，使用八爪鱼平台的登录账号和密码来获取全新 Access Token。

在 API 请求中通过 `Authorization` 请求头携带 Token：

```
Authorization: Bearer YOUR_ACCESS_TOKEN
```

## Token 刷新

Access Token 过期后，使用 [刷新 Token](/en/api-reference/access-token/refresh-token) 接口获取新的 Access Token。

## 注意事项

- 请将 Access Token 安全存储，切勿公开发布
- 使用环境变量或密钥管理器管理凭据，切勿硬编码在代码中
- 如怀疑 Token 已被泄露，请从八爪鱼账户设置中重置
