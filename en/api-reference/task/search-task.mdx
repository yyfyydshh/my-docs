# Search Task

> **Required plan:** Standard, Professional, or Enterprise

## Endpoint

```
GET https://openapi.octoparse.com/task/search
```

## Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| `taskGroupId` | string | No | Filter by task group ID |
| `keyword` | string | No | Search by task name |

## Response

Returns a list of matching tasks with their IDs, names, and statuses.
