# Octoparse CLI

> Octoparse CLI 是八爪鱼采集器的命令行客户端。用于在终端、脚本或 CI/CD 流水线中管理任务、触发云采集并导出数据，无需打开客户端或浏览器控制台。

## 适用场景

- **自动化与定时任务** — 在 cron、GitHub Actions、Jenkins 等环境中按计划运行采集任务。
- **本地开发与调试** — 在不离开终端的情况下查询任务状态、运行示例、快速导出数据。
- **批量操作** — 通过脚本批量启动、停止任务或导出数据，结合 `jq` 等工具做后处理。
- **与 OpenAPI 互补** — 当你只需要少量常用操作时，CLI 比直接调用 [OpenAPI](/en/api-reference/intro) 更轻量。

## 前置条件

- **Node.js 18 或更高版本** — CLI 通过 npm 安装。
- **Octoparse 账户** — 在 [账户中心](https://www.octoparse.com/console/account-center/api-keys) 获取 API Key。

## 安装

```bash
npm install -g @octoparse/cli
```

安装完成后，验证版本：

```bash
octoparse --version
```

## 认证

使用你的 API Key 登录。凭证将保存在本地配置文件中，后续命令自动读取。

```bash
octoparse auth login --api-key YOUR_API_KEY
```

在 [Octoparse 账户中心](https://www.octoparse.com/console/account-center/api-keys) 获取你的 API Key。

如需切换账户或退出登录：

```bash
octoparse auth logout
```

## 常用命令

| 命令 | 说明 |
| --- | --- |
| `octoparse tasks list` | 列出当前账户下的所有任务 |
| `octoparse tasks search <query>` | 按名称搜索任务 |
| `octoparse run <task-id>` | 启动一次云端采集 |
| `octoparse status <run-id>` | 查看某次运行的状态 |
| `octoparse export <run-id>` | 从已完成的运行中导出数据 |
| `octoparse templates search <query>` | 搜索预置模板库 |

## 示例

### 搜索模板

```bash
octoparse templates search amazon
```

### 启动任务并等待完成

使用 `--wait` 阻塞命令直到任务完成，便于在脚本中串联后续步骤。

```bash
octoparse run TASK_ID --wait
```

### 以 JSON 格式导出数据

```bash
octoparse export RUN_ID --format json --output ./data.json
```

支持的格式包括 `json`、`csv` 和 `xlsx`。

### 在 CI 中按计划运行

下例展示如何在 GitHub Actions 中每日运行采集并把结果作为构件上传。

```yaml
name: Daily scrape
on:
  schedule:
    - cron: "0 2 * * *"
jobs:
  scrape:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/setup-node@v4
        with:
          node-version: "20"
      - run: npm install -g @octoparse/cli
      - run: octoparse auth login --api-key ${{ secrets.OCTOPARSE_API_KEY }}
      - run: octoparse run ${{ vars.TASK_ID }} --wait
      - run: octoparse export latest --format csv --output ./data.csv
      - uses: actions/upload-artifact@v4
        with:
          name: scrape-output
          path: ./data.csv
```
