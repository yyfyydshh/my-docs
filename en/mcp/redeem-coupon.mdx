# redeem_coupon_code

Redeem a promotional or resource coupon code through the MCP interface.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `code` | string | The coupon or promotional code |

## Response

Returns redemption status and any credits or benefits applied.

<Note>
  Coupon redemption through MCP is available for all Octoparse account types. Coupon eligibility depends on the specific offer.
</Note>
