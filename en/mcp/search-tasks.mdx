# search_tasks

List and search your existing Octoparse tasks by name or criteria.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `query` | string | Search by task name |
| `status` | string | Filter by status (active, paused, draft) |

## Response

Returns matching tasks with task ID, name, status, last run time, and cloud/local mode.
