# export_data

Export data from a completed Octoparse task run.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `taskId` | string | Task ID |
| `runId` | string | Run ID (optional if task has recent run) |
| `format` | string | Export format: json or csv |
| `limit` | number | Maximum records to return |

## Response

Returns the extracted data in the requested format.
