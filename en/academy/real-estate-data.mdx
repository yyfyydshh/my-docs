# Real Estate Data Extraction

> How real estate teams collect listing, rental, sold, agent, and neighborhood data from public portals while handling freshness, duplicates, and compliance.

## What Real Estate Teams Extract

| Data Type | Use Case | Typical Fields |
|---|---|---|
| Property listings | Inventory monitoring | Price, address, beds, baths, sqft, photos, listing date |
| Rental listings | Rental market analysis | Rent, beds, baths, location, amenities, available date |
| Sold data | Market trend analysis | Sale price, sale date, days on market, price history |
| Agent data | Lead generation | Agent name, brokerage, phone, listings, reviews |

## Common Sources

- Zillow, Redfin, Realtor.com (US)
- Rightmove, Zoopla (UK)
- Domain, Realestate.com.au (Australia)

## Key Metrics to Track

- **Days on market** - Indicator of demand
- **Price changes** - Negotiation signals and market movement
- **Price per sqft** - Normalized comparison metric
- **New listings** - Market activity
