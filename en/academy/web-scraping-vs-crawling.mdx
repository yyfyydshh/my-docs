# Web Scraping vs Web Crawling

> Scraping and crawling are often confused but solve different problems. Learn the technical and intent-level differences, and when to use each.

## Definitions

**Web Scraping** focuses on extracting specific, structured data from targeted pages. You know what data you want before you start.

**Web Crawling** systematically browses the web to discover and index pages. The goal is to map, index, or traverse a site.

## Comparison

| Aspect | Scraping | Crawling |
|---|---|---|
| Goal | Extract specific data fields | Discover and index pages |
| Scope | Targeted URLs and pages | Broad site traversal |
| Output | Structured data | Index, sitemap, page content |
| Volume | Moderate to high, focused | Often very high, broad |

## When to Use Which

- **Scraping** for: Price monitoring, lead generation, competitive intelligence
- **Crawling** for: Building search indexes, site auditing, finding all pages on a domain
